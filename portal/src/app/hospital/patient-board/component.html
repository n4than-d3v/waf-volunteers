Tutorial

Hardware

At the start of each day, locate the corresponding remote control for each board, then press the power button in the top left corner to turn them on.
At the end of the day, use the same button to turn each board off; then leave the remote in a visible place for the morning shift.

Each board has a corresponding mouse. Use the mouse cursor to navigate the board. If moving the mouse doesn't work, try left-clicking once - this will "wake it up". If this doesn't work, you may need to change the battery.
There are buttons on the left of the mouse - please avoid pressing these when holding, as it may cause the screen to refresh.

If you're experiencing technical difficulties, or see any errors, locate this plug. Turn it off, wait 10 seconds, then turn it on again.
If this doesn't solve the issue, contact the office.

Board selection

After the TV is turned on, you will be presented with a list of "boards" to choose from.
Each board shows different areas of the hospital, relevant to the board's intended purpose. E.g. "SU4" will only show the patients in SU4.
Simply click on the board you wish to view, and you will be taken to it.

Switching between boards

If you are viewing a board, and would like to change the board currently being displayed, click the "Change board" button in the top left of the screen.
This will take you back to the board selection screen.
With this, you have access to information regardless of your current location in the hospital.
E.g. if you're in top barn and wish to check if there's any FFs in SU4, you can do so without needing to walk there.

Selecting your shift

When you are viewing a board, it's important that you ensure your shift is selected (highlighted in purple).
When switching shifts, you will notice the board updates with the relevant feeding times for your shift. If the morning shift is selected, you won't see afternoon or evening feeds, and vice versa.
To select your shift, simply click it; you should see it turn purple shortly afterwards.

Filtering

If you wish to view the boards in traditional "whiteboard-style", that is perfectly fine, and it's configured to operate that way by default.
If, however, you wish for the boards to be more dynamic, you can use the filters available to customise what gets displayed to you.

:

Summary

Unselected by default.
If selected, it will group the patients on the selected board by their species (and age), showing a summary of the food that needs to be made up, as well as listing all pens where that species is located.

References

Unselected by default.
If selected, you will see patient references. These are used to identify individual patients on the vet system and patient cards.

No feeds

Selected by default.
If selected, you will see patients that do not require feeding on the selected shift. If unselected, you won't.

Needs cleaning

Selected by default.
If selected, you will see pens that have no patients in and require a thorough cleaning. If unselected, you won't.

Needs setting up

Selected by default.
If selected, you will see pens that have no patients in and require setting up. If unselected, you won't.

Ready to use

Unslected by default.
If selected, you will see pens that have no patients in and have been set up, so are ready to use. If unselected, you won't.

Completed

Selected by default.
If selected, you will see patients that are considered "completed" on the selected shift. If unselected, you won't.

Feed at

Unselected by default.
If selected, you will only see patients that have a feeding at a time you specify using the drop-down boxes. Use this if you only wish to see patients that require feeding at a specific time on your shift.

Using the summary

The summary works best for teams that operate by assigning individuals to particular species, rather than areas.
It will tell you how much food to prepare for a species (and age) on any given board, as well as list the pens they are located in.
When listing the pens, it will show you how many patients are located in each pen (if there is an uneven split), as well as whether the pen has been marked as completed or not.
The summary is dynamic, so if you apply filters, the summary will update to reflect the filters you've applied, in the same way as the pens.
It's important to remember it will only summarise patients on the selected board. Other boards are excluded.
If you wish to see a summary of both the "Top Barn" + "Hospital 2" boards, use the "Everything" board.
